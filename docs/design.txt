PROBLEM STATEMENT: Write a restaurant menu and total payment program.
Welcome the user to the restaurant.
In a  loop ask the user to input a menu item or stop to quit. 
Each menu item should have a price that is added up. When the user types stop, add up their item total and then display the total. 
After displaying the item total ask for the tip percentage, calculate the tip percentage and then display their total bill with a price breakdown. (just like a receipt)
Then display a thank you and goodbye.

NOUNS: restaurant,menu item, price, item total, total bill
VERBS: welcome, ask, input, add, display, calculate, breakdown


DEFINING DIAGRAM:
INPUTS: menu item, stop, tip percentage

PROCESSING:
Prompt user for input
IF input is menu item, add to total, continue loop
IF input is stop, terminate loop 
Display user input
Display total without tip
Ask for tip percentage
Calculate tip and total bill
Display total bill 

OUTPUTS:
Display user input
Display total without tip
Display total with tip
